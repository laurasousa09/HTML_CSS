Perguntas Frequentes

O que é HTML?

HTML é a linguagem de marcação usada para estruturar páginas na web.

O que é CSS?

CSS é a linguagem de estilo usada para definir o visual das páginas HTML.

O que significa “semântica” no HTML?

Refere-se ao uso de tags com significado próprio, que indicam a função do conteúdo.
